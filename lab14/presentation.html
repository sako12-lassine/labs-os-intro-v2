Лабораторная работа №14

1. Цель работы

Изучение основ программирования в оболочке ОС UNIX.

2. Задачи

  • Реализация механизма семафоров
  • Эмуляция команды man
  • Генерация случайных последовательностей

3. Ход выполнения работы

3.1. Семафоры (semaphore.sh)

```bash #!/bin/bash SEMAPHORE=“/tmp/semaphore.lock” for i in {1..3}; do while [ -f “$SEMAPHORE" ]; do sleep 2; done touch "$SEMAPHORE” echo “Ресурс захвачен” sleep 2 rm -f “$SEMAPHORE” done

3.2. Эмуляция man (my_man.sh)

```bash #!/bin/bash zcat “/usr/share/man/man1/$1.1.gz” | head -30

3.3. Случайные буквы (random_letters.sh)

```bash #!/bin/bash for i in $(seq 1 ${1:-10}); do printf "\\x$(printf “%x” $((97 + (RANDOM % 26))))” done echo “”
