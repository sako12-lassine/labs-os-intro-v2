Лабораторная работа №11

1. Цель работы

Освоение работы с текстовым редактором Emacs.

2. Задачи

  • Установка Emacs
  • Создание bash-скрипта lab11.sh
  • Редактирование и сохранение файла
  • Выполнение скрипта

3. Ход работы

  • Установка: sudo apt install emacs-nox
  • Запуск: emacs -nw
  • Создание файла: C-x C-f → lab11.sh
  • Ввод bash-скрипта
  • Сохранение: C-x C-s
  • Выход: C-x C-c
  • Выполнение: chmod +x lab11.sh && ./lab11.sh

4. Результаты

Скрипт выводит: Hello World

5. Выводы

Основные commandes Emacs: - C-x C-f — открыть/создать - C-x C-s — сохранить - C-x C-c — выход

6. Заключение

Лабораторная работа выполнена в полном объёме.
