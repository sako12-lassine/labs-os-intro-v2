# Лабораторная работа №13

## 1. Цель работы
Освоение работы с текстовым редактором emacs.

## 2. Основные команды

| Команда | Действие |
|---------|----------|
| `C-x C-f` | Открыть/создать файл |
| `C-x C-s` | Сохранить |
| `C-x C-c` | Выход |
| `C-k` | Вырезать строку |
| `C-y` | Вставить |
| `C-space` | Начать выделение |
| `C-w` | Вырезать выделенное |
| `M-w` | Копировать |
| `C-/` | Отменить |
| `C-s` | Поиск |
| `M-%` | Замена |
| `C-x 2` | Разделить окно |
| `C-x 3` | Разделить вертикально |
| `C-x 1` | Оставить одно окно |
| `C-x b` | Переключить буфер |
| `C-x C-b` | Список буферов |
| `C-a` | Начало строки |
| `C-e` | Конец строки |
| `M-<` | Начало файла |
| `M->` | Конец файла |

## 3. Режимы emacs

- **Командный режим** — для навигации и команд
- **Режим вставки** — для ввода текста
- **Режим последней строки** — для сохранения и выхода

## 4. Буферы и окна

- **Буфер** — область памяти с текстом
- **Окно** — область экрана, отображающая буфер
- Можно открыть любое количество буферов
