# Лабораторная работа №8

## 1. Цель работы
Освоение grep, find, jobs, kill, ps.

## 2. Задачи
- Перенаправление ввода-вывода (>, >>, |)
- Поиск файлов (find)
- Фильтрация текста (grep)
- Управление процессами (&, jobs, kill)

## 3. Ход работы
- ls /etc > file.txt
- grep ".conf" file.txt
- find /var/log -name "*.log" > logfile &
- ps aux | grep nano
- kill -9 PID

## 4. Результаты
- Найдены файлы .conf
- Процесс nano запущен в фоне и завершён

## 5. Выводы
Поиск и управление процессами освоены.
