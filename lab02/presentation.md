# Лабораторная работа №2

## 1. Цель работы
Освоение базовых команд git, создание SSH и GPG ключей.

## 2. Задачи
- Установка git и gh
- Генерация SSH-ключа (ed25519)
- Генерация GPG-ключа (RSA 4096)
- Настройка подписи коммитов

## 3. Ход работы
- ssh-keygen -t ed25519
- gpg --full-generate-key
- git config --global user.signingkey
- git commit -S -m "message"

## 4. Результаты
- SSH и GPG ключи добавлены в GitHub
- Подписанные коммиты отображаются как Verified

## 5. Выводы
Git настроен. Коммиты подписываются автоматически.
