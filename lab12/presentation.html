5. Основные команды Bash

  • $# - количество аргументов
  • $@ - все аргументы
  • $0 - имя скрипта
  • $? - код завершения
  • -d - проверка каталога
  • -f - проверка файла
  • -r - проверка чтения
  • -w - проверка записи
  • -x - проверка выполнения
