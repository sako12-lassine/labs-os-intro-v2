# Лабораторная работа №1

## 1. Цель работы
Приобретение практических навыков установки Linux на виртуальную машину.

## 2. Задачи
- Создание виртуальной машины в QEMU/KVM
- Установка Fedora 41 Server
- Настройка пользователя и сети
- Анализ загрузки через dmesg

## 3. Ход работы
- Создан образ диска (60 ГБ)
- Установлена Fedora 41 Server
- Настроены SELinux, сеть, пользователь lsako

## 4. Результаты
| Параметр | Значение |
|----------|----------|
| Версия ядра | 6.11.4 |
| Гипервизор | KVM |
| Память | ~2,9 ГБ |

## 5. Выводы
