# Лабораторная работа №9

## 1. Цель работы
Освоение работы с Midnight Commander (mc).

## 2. Задачи
- Запуск mc и изучение интерфейса
- Операции с файлами (F5, F6, F7, F8)
- Встроенный редактор (F4)

- mc (запуск)

- F4 (редактирование)

## 4. Результаты
- Файлы скопированы и перемещены
- Редактор mc позволяет править текст
