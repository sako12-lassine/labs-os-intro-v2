# Лабораторная работа №6

## 1. Цель работы
Освоение базовых команд Linux.

## 2. Задачи
- Навигация: cd, pwd
- Просмотр: ls, ls -la, ls -alF
- Создание/удаление: mkdir, rmdir, rm -r
- Справка: man history

## 3. Ход работы
- ls /etc > file.txt
- grep ".conf" file.txt > conf.txt
- find ~ -name "c*"

## 4. Результаты
- Созданы и удалены каталоги
- Найден каталог /var/spool/cron

## 5. Выводы
